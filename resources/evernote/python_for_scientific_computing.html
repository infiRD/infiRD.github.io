Spyder [5] - matlab like
 - dokumentacia tu [4], napisany v pythone, ma 1800* na githube [5] a kontinualny, silny vyvoj k 01/2017 [6]

Rodeo [42][41] - matlab like
 - nova vec, napisany v JS, na githube ma 3300* [42]. zaujimave ze ma viac * ako spyder, ale kratsi a slabsi vyvoj .. 

Jupyter notebook [25] - pozri osobitnu poznamku
 - klient - server architektura [29] s moznostou pracovat s roznymi kernelmi ako je iPython [25], iJulia [30], iHaskell [31].. 
 - na komunikaciu server <-> kernel sa pouziva zeroMQ
 - podporuje Latex komentare, pekne dokumentovanie kodu on-the-fly .. 
 - ma nbviewer [46] servis pomocou ktoreho ludia zdelaju cele KNIHY! [48]
 - github vie renderovat Jupyter notebooky (2015) .. [47]

PyCharm [8]
 - IDE od developerov InteliJ, maju kod na githube [9] 4000*, ale to je pre cele intelij ..
 - integrates with IPython notebook, supports Anaconda as well as other scientific packages like matplotlib and NumPy [13]
 - vie sa pripojit na jupyter, funguje celkom bezproblemovo s lokalnym pytonom v anaconde ...
 - nekompatibilna cell syntax ## so spyderom ! #%%
